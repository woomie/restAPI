Project Overview

This API allows users to manage their expenses and income in a system. It supports basic CRUD (Create, Read, Update, Delete) operations for both income and expense records. The API is designed to handle user data related to financial tracking, including savings, payment obligations, and housing expenses.

Base Url

http://localhost:4000/

Installation

Get/income/:id

Retrieve a specific income by id

  • URL:/income/:id
  • Method:GET
  • Parameters: id, the unique id of the income record
  • Response:
                    {
                        "interest": 200,
                        "others": 100,
                        "secondary_income": 1500,
                        "support_payment": 0,
                        "wages": 5000
                      }

Post/income

create new income

  • URL:/income
  • Method:POST
  • Request body:

                        "support_payment": 0,
                        "wages": 5000
